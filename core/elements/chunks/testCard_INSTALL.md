# Установка чанка testCard в MODX

## Способ 1: Через MODX Manager (рекомендуется)

1. Войдите в MODX Manager
2. Перейдите в **Elements** → **Chunks**
3. Нажмите правой кнопкой мыши на папку **Chunks** → **Create** → **New Chunk**
4. Заполните поля:
   - **Chunk name (required)**: `testCard`
   - **Description**: `Карточка теста для списка (с поддержкой статусов)`
   - **Category**: выберите подходящую категорию или создайте новую "TestSystem"
5. Скопируйте содержимое файла `testCard.tpl` в поле **Chunk code**
6. Нажмите **Save**

## Способ 2: Через SQL

```sql
-- Найдите ID категории или создайте новую
INSERT INTO `modx_categories` (`category`, `parent`)
VALUES ('TestSystem', 0);

-- Вставьте чанк (замените CATEGORY_ID на реальный ID категории)
INSERT INTO `modx_site_htmlsnippets` (`name`, `description`, `snippet`, `category`)
VALUES (
    'testCard',
    'Карточка теста для списка (с поддержкой статусов)',
    '<!-- Содержимое из testCard.tpl -->',
    CATEGORY_ID
);
```

## Использование

После создания чанка используйте его в сниппете getTestInfoBatch:

```
[[!getTestInfoBatch?
    &parents=`56`
    &depth=`3`
    &showAllStatuses=`1`
    &tpl=`testCard`
]]
```

## Что отображает чанк

1. **Название теста** - кликабельная ссылка
2. **Бейдж статуса**:
   - **Черновик** (желтый) - только для LMS Admins и LMS Experts
   - **Приватный** (серый) - для всех, кто имеет доступ
   - **Публичный** (зеленый) - только для LMS Admins и LMS Experts
3. **Краткое описание** - из поля introtext
4. **Информация**: количество вопросов и проходной балл
5. **Кнопки действий**:
   - "Начать тест" - для всех
   - "Редактировать" - если есть права на редактирование
   - "Управление доступом" - если можно управлять доступом

## Кастомизация

Вы можете изменить:
- Цвета бейджей в блоке `<style>`
- Макет карточки (bootstrap классы)
- Иконки (bootstrap icons)
- Дополнительные поля из плейсхолдеров

## Доступные плейсхолдеры

Полный список плейсхолдеров см. в комментариях внутри файла `testCard.tpl`.
